# 🧮 PalkApp — Estonian Salary Calculator

**PalkApp** is a simple mobile application that helps users calculate their **net salary in Estonia** based on their **gross income**.  
It supports multiple languages and shows a clear **visual breakdown of taxes** and contributions.

---

## 🌍 Features

- 🌐 Multi-language interface:
  - 🇷🇺 **Russian**
  - 🇬🇧 **English**
  - 🇪🇪 **Estonian**
- 💰 Calculates:
  - **Net salary (after taxes)**
  - **Income tax**
  - **Pension pillar (II)**
  - **Unemployment insurance (worker + employer)**
  - **Total employer cost**
- 📊 Displays a **pie chart** of salary distribution
- 🧾 Based on **Estonian tax system (2025)**
